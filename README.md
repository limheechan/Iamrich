# Iamrich

간단하게 연습해본 첫번쨰 APP 입니다. 
해당 앱으로 Imgview 와 Label 의 속성 인스펙터 에 대해 알 수 있었습니다. 
또한, 아이콘 제작 과 에셋 안의 JSON 을 통해 각이미지가 자동적으로 자신의 자리에 맞게 갈 수 있다는것도 알게되었습니다.
